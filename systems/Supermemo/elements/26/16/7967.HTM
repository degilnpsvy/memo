9.2 Regular Expressions

A regular expression (also known as a regexp or regex) describes a textual pattern.

Ruby’s Regexp class(* JavaScript programmers should note that the Ruby class has a lowercase e, unlike the JavaScript RegExp class.) implements regular expressions, and both Regexp and String define

pattern matching methods and operators. Like most languages that support regular

expressions, Ruby’s Regexp syntax follows closely (but not precisely) the syntax of Perl

5.
