You can compare both the HEAD and MERGE_HEAD versions against the working directory (“merged”) version:

$ git diff HEAD

diff --git a/hello b/hello

index e63164d..4e4bc4e 100644

--- a/hello

+++ b/hello

@@ -1,3 +1,7 @@

hello

+<<<<<<< HEAD:hello

worlds

+=======

+world

+>>>>>>> alt:hello

Yay!

And then this:

$ git diff MERGE_HEAD

diff --git a/hello b/hello

index 562080a..4e4bc4e 100644

--- a/hello

+++ b/hello

@@ -1,3 +1,7 @@

hello

+<<<<<<< HEAD:hello

+worlds

+=======

world

+>>>>>>>

alt:hello

Yay!
