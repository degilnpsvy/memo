字符串比较、算术比较和与文件有关的条件测试。

string1 = string2 如果两个字符串相同则结果为真

string1 != string2 如果两个字符串不同则结果为真

-n string 如果字符串不为空则结果为真

-z string 如果字符串为null（一个空串）则结果为真

expression1 -eq expression2 两个表达式相等则结果为真

 

-ne 不等

-gt 大于

-ge 大于等于

-lt 小于

-le 小于等于

! expression 为假

 

-d file 是目录则为真

-e file 文件存在

-f file 普通文件

-g file  如果文件的set-group-id位被设置则结果为真

-r file 可读

-s file 文件大小不为0

-u file 如果文件的set-user-id位被设置则结果为真

-w file 文件可写

-x file 文件可执行
