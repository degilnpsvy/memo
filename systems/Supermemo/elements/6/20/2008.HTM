Functions are defined as lists of the above entities, usually as lists of calls to other, existing functions.

All functions have return values (as with Perl functions and non-void Java methods); a function's return value is simply the value of the last item in the list, usually the value returned by the last function called.

A function call within another function is equivalent to a statement in other languages, and we use statement interchangeably with function call in this chapter. Here is the syntax for function: (function-name argument1 argument2 ...)
