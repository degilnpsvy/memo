Org knows ordered lists, unordered lists, and description lists.

Unordered list items start with ‘-’, ‘+’, or ‘*’ as bullets.

Ordered list items start with a numeral followed by either a period or a right parenthesis15, such as ‘1.’ or ‘1)’16. If you want a list to start with a different value (e.g., 20), start the text of the item with [@20]17. Those constructs can be used in any item of the list in order to enforce a particular numbering.

Description list items are unordered list items, and contain the separator ‘ :: ’ to distinguish the description term from the description.~
