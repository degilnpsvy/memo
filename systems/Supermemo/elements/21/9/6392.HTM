Git is simple to use. How to make git lists its options and the most common subcommands?

Just type git. Without any arguments, Git lists its options and the most common subcommands.

$ git

git [--version] [--exec-path[=GIT_EXEC_PATH]] [-p|--paginate|--no-pager] [--bare] [--git-dir=GIT_DIR] [--work-tree=GIT_WORK_TREE] [--help] COMMAND [ARGS]
