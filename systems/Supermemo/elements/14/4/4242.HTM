1.3.1 termlnx, termwin.exe

This is a tiny terminal program transmitting data through TCP socket. It can be used to connect to the UART to TCP wrapper in RLX-VM. Termlnx is for Linux, and termwin.exe is for Windows.

It can be simply used as

termlnx host port

where host is the host name or the IP address where the UART wrapper is, port is the TCP number.

There are two modes for this terminal program: default mode and line echo mode. In default mode, the program sends every character from keyboard immediately after it’s keyed in. The characters are not echoed locally. In line echo mode, characters are only sent when there is a newline. In other words, user can use backspace to modify the command before it’s sent.
