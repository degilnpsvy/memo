2.4 Post Simulation Analysis

If the simulation lasts a long time, user is free to stop the simulation by “CTRL-C”. When the simulation stops, some performance statistic data is printed out. If the simulation stops for the 2nd “CTRL-C” for some rare case, the performance information may not show.

Simulation view data indicate the performance of the target system. Note that RLX-VM is not cycle accurate. The information here is only for your reference.

Simulation View (Performance of the SOC):

Time spent xxx:

Shows the time passed in simulation’s view, which indicates the time the program will cost if it runs on real hardware once it’s ready.

[Processor Name] Instruction Count xxx, yyy (inst/sec)

Shows the instructions run on the specific processor, and the performance (counted as instruction per second) of the processor in simulation’s view.

Total Instruction Count xxx

Shows the total instructions run on all the processors.

Total system performance

Shows the performance of the target system.
