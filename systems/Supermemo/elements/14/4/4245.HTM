2.1 Installed Directory Structure

Figure 2-1 Directory Structrue

The installed directory structure is shown in Figure 2-1.

The bin directory contains binary files, including the wrapper script to run RLX-VM, tools for encrypted configure form support.

The doc directory contains the user guide, introduction slide, and documents which describe the architecture of the simulated platform.

The example directory contains the source code that user can follow to develop new model. Most example models are installed with the development package.

The include directory contains the header files for developing purse. It is installed with the development package.

The lib directory contains the libraries of component models in RLX-VM.

The platform directory contains the example configure forms to feed RLX-VM. Subdirectory others contains example configure forms to show the usage of some components. Subdirectory rlxocp contains configure forms for Realtek OCP reference platform. Subdirectory rlxocp-1.1 contains configure forms for Realtek OCP reference platform v1.1.
