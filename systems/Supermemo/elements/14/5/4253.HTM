2.5.3 Global Configuration Parameters

Configuration parameters below are global parameters:

l SIM_RESOLUTION

 

Type: string

Description: This parameter defines the simulation resolution. Any time value that is smaller than the simulation resolution is truncated. Refer to SystemC’s Language Reference Manual (LRM) for detail. The value can be “40 NS”, namely frequency is 25MHz. The first part of the string is a double value, and the second part is a time unit, which can be “FS”, “PS”, “NS”, “US”, “MS”, “SEC”.

l SHOW_TLM

 

Type: bool

Default: false

Description: Set this parameter to true to turn on showing OCP transaction for all model instances. If you want to see OCP transaction of a specific model instance, set the parameter of that model instance.

l SHOW_ACTION

 

Type: integer

Default: 0

Description: Set this parameter to a positive value to tell the module to output the internal action information of all model instances. If you want to see the internal action of some specific model instance, set the parameter of that model instance. See Table 2-1 for predefined message level.

l COLORFUL_MESSAGE

 

Type: bool

Default: true

Description: Set true to enable colorful message. Yellow for warning message; red for error message.
