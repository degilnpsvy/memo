The Index register low bits hold the TLB index. ( Ancient R2000-compatible CPUs had a different register layout, with the index offset, starting from bit 8 upward. You’re unlikely to meet one of those.) Not many bits are needed, since no MIPS CPU has yet had a TLB with more than 128 entries. The high bit (bit 31) is magic, being set by tlbp when the probe fails to finds a matching entry. Bit 31 is a good choice—because it makes the value appear negative, it’s easy to test for.
