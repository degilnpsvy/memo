If sequential instruction execution crosses from Non-secure memory to Secure memory, then if the Secure memory entry point contains an SG instruction and the whole of the instruction at the Secure memory entry point is flagged as Secure and Non-secure callable, it is CONSTRAINED UNPREDICTABLE whether:
  • The PE changes to Secure state.
  • A fault is generated:
    ― An INVEP SecureFault in a PE with the Main Extension.
    ― A Secure HardFault in a PE without the Main Extension.
