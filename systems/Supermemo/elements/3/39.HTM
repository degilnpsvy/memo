Title: Keyboard tips

  • Enter - expand or collapse a branch
  • Ins - insert an element at current selection
  • Del - delete selected branch
  • Ctrl+Up - go to the parent of the current element
  • Ctrl+D - dismiss the element
  • Ctrl+F - search the branch
  • Ctrl+L - learn the selected branch
  • Ctrl+Shift+B - show branch statistics
