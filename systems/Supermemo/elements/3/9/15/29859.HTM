The pseudocode hierarchy for a load or store instruction is as follows:
• the instruction operation uses the [...] helper function
• memory attributes are determined from the default system address map or using an MPU as defined in The system address map on page B3-704 or Protected Memory System Architecture, PMSAv7 on page B3-761 respectively.
• the access is governed by whether the access is a read or write, its address alignment, data endianness and memory attributes.
