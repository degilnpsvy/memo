The VTOR characteristics are:
Purpose: Holds the vector table address for the selected Security state.
Usage constraints: Privileged access permitted only. Unprivileged accesses generate a BusFault.
  This register is word accessible only. Halfword and byte accesses are UNPREDICTABLE.
Configurations: This register is always implemented.
Attributes: 32-bit read/write register located at 0xE000ED08.
  Secure software can access the Non-secure view of this register via VTOR_NS located at 0xE002ED08. The location 0xE002ED08 is RES0 to software executing in Non-secure state and the debugger.
  This register is banked between Security states.
