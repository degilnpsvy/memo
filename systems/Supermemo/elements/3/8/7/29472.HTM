Mouse shortcuts

  • Click - edit component (e.g. to edit text)
  • Shift+Click - Edit file (e.g. to edit pictures)
  • Ctrl+Click - View file (e.g. to view a picture in the default viewer or view an HTML in the default browser)
  • Ctrl+Shift+Click - drag component to ClipBox
  • Alt+Click - switch components (or element) between presentation, editing and dragging

Single keys

  • A - Edit the first answer
  • E - Edit texts
  • Q - Edit the first question
  • Del - Delete the currently displayed element (use Shift+Ctrl+Del when in editing mode)
  • Ins
    • in the contents window: insert element
    • in the browser: select or deselect the element
    • in the schedule manager: insert a new activity in the currently selected row
  • Tab
    • in dialogs: move to the next button, checkbox, etc.
    • in the element window: move to the next component

Function keys

  • F1 - Help
  • F2 - Translated help
  • F3
    • in text components: search for a string
    • in element window, contents window, element browser, registries, tasklist manager: Repeat last search
  • F4 - Open the tasklist manager with the currently used tasklist
  • F5 - Apply warrior layout without making it the default layout
  • F6 - Filter HTML or RTF source (e.g. to remove formatting that makes incremental reading difficult)
  • F7 - Spell-check
  • F8 - Display encoding codes of the registry member name
  • F9 - View the file associated with the component. For example, if you use MS Office, F9 on rich text components will start editing your texts in MS Word, and F9 on HTML components will use your default browser to view your HTML texts (see also: Ctrl+F9 for editing files)
  • F11
    • Random review in collection or browser
    • Random jump/pick in a registry or tasklist manager
  • F12 - Sleep and learning timeline (Tools : Sleep Chart)

Shift+key

  • Shift+Enter
    • in HTML components: single-spaced new line in HTML (tag <br>); Enter results in double-spacing (tag <p>)
    • in the contents window: View the currently focused element; Enter can also be used if the element has not children
  • Shift+Tab - Move to the previous option or option group
  • Shift+F2 - Search SuperMemo Wiki Help
  • Shift+F3 - Search for leeches
  • Shift+F4 - Import mail from MS Outlook for incremental reading
  • Shift+F12 - Quick backup

  • Ctrl+] - increase font size
  • Ctrl+[ - decrease font size
  • Ctrl+Del
    • in the element window, delete the current component
    • in the schedule manager, delete the currently selected activity
    • in the tasklist manager, delete the currently selected task
  • Ctrl+F2 - Execute the final drill
  • Ctrl+F3 - Search the web for the currently selected phrase (Google, Wikipedia, dictionaries, etc.)
  • Ctrl+F5 - Restore the default windows layout
  • Ctrl+F6 - Next window
  • Ctrl+F7 - Set a read-point in an article
  • Ctrl+F8 - Download remote images referred to in an HTML component
  • Ctrl+F9 - Edit the file associated with the component. For example, Ctrl+F9 on image components will start editing your images in Paint

  • Ctrl+F11
    • Random learning

  • Ctrl+A

  • Ctrl+C - Copy to the clipboard
  • Ctrl+D

  • Ctrl+E
    • in the element window: enter the editing mode

    • in the tasklist manager: edit the currently selected task
  • Ctrl+F - Find elements (in the main window) or Find (in other windows)
  • Ctrl+G
    • go to an element with a given number (see the caption of element data window)

  • Ctrl+I

  • Ctrl+M

  • Ctrl+N - Add a new article to the current category by pasting the text from the keyboard
  • Ctrl+O
    • File : Reopen collection
    • in the schedule manager: open a plan
  • Ctrl+P - Tools : Plan - plan your daily schedule
  • Ctrl+Q - Import file (in a component)(e.g. to add sound to a sound component)
  • Ctrl+R
    • in the element window: Forget - reset the learning process of an element and move it back to the pending queue

    • in the registry window: Find and replace
  • Ctrl+S

    • in the tasklist manager: sort the tasklist for priority
    • in the schedule manager: save changes to the current schedule
  • Ctrl+T
    • Edit next component (e.g. quickly switch between the question and the answer)

    • in the schedule manager: Split the currently selected activity
  • Ctrl+U - Underline font in text components (if applicable)
  • Ctrl+V - Paste from clipboard
  • Ctrl+W - Tools : Workload
  • Ctrl+X - Cut
  • Ctrl+Y - Redo last Undo (in HTML text editing)
  • Ctrl+Z - Undo text editing (same as Alt+Backspace)

Ctrl+Alt+key

Important change in SuperMemo 15: To avoid interference with AltGr on various international keyboards, most of Ctrl+Alt shortcuts have been removed from SuperMemo 15. Some of the most often used combinations have been replaced with new shortcuts. Experience shows that with heavy use you can transition to new shortcuts within weeks. The replacement shortcuts have been chosen to be mnemonic and resemble old shortcuts wherever possible. If you are unhappy with any of the changes, please place your suggestions on the Talk page or at SuperMemoPedia. New commands can always be made available from the Commander:

  • Ctrl+Alt+Up - Drag up all components in dragging mode (in the element window)
  • Ctrl+Alt+Left - Drag left all components in dragging mode (in the element window)
  • Ctrl+Alt+Right - Drag right all components in dragging mode (in the element window)
  • Ctrl+Alt+Down - Drag down all components in dragging mode (in the element window)
  • Ctrl+Alt+Enter - Collapse the current branch (in the contents window)
  • Ctrl+Alt+F10 - Show or hide the background behind SuperMemo
  • Ctrl+Alt+F11 - Show or hide the toolbars window
  • Ctrl+Alt+F12 - Move one difficulty level up (see: Ctrl+Alt+Shift+F12)
