There are three general MPU registers:
• The MPU Type Register specified in MPU Type Register, MPU_TYPE on page B3-766. This register can be used to determine if an MPU exists, and the number of regions supported.
• The MPU Control Register specified in MPU Control Register, MPU_CTRL on page B3-767. The MPU Control register includes a global enable bit that must be set to 1 to enable the MPU.
• The MPU Region Number Register specified in MPU Region Number Register, MPU_RNR on page B3-769.
