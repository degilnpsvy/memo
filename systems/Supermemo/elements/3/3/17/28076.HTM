Write a function that removes each 9 that it is in between 7s.

seven_ate9('79712312') => '7712312'
seven_ate9('79797') => '777'

Input: String Output: String

 

import re

def seven_ate9(str_):
    return re.sub(r'79(?=7)', r'7', str_)
