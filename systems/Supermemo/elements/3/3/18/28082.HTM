Description:

In this Kata you will create a function that takes a list of lists as an input and returns a flat list.

Example:

flatten [[1,2],[3,4]] == [1,2,3,4]
flatten [[1],[2],[3],[4]] == [1,2,3,4]
 
 
def flatten(list):
  return [item for sublist in list for item in sublist]
 
 
from itertools import chain

def flatten(l):
  return list(chain(*l))
 
 
def flatten(lists):
    flat_list = []
    for list in lists:
        flat_list += list
    return flat_list
 
 
def flatten(list):
  result = []
  for j in list:
      for i in j:
          result.append(i)
  return result
