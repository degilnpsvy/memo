The NSACR characteristics are:
Purpose: Defines the Non-secure access permissions for both the FP Extension and coprocessors CP0 to CP7.
Usage constraints: Privileged access permitted only. Unprivileged accesses generate a BusFault. This register is word accessible only. Halfword and byte accesses are UNPREDICTABLE.
Configurations: This register is always implemented.
Attributes: 32-bit read/write register located at 0xE000ED8C. This register is RAZ/WI when accessed as Non-secure. This register is not banked between Security states.
