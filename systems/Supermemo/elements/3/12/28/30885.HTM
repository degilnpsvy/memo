IFPJD The maximum supported number of external interrupts is 496, regardless of whether the Main Extension is implemented.
IQWTM For exceptions with software configurable priority numbers, software configures the priority numbers by using registers SHPR1 - SHPR3 in the System Control Block (SCB).
RNFSM Software configurable priority numbers start at 0.
