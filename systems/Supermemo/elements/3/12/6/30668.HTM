B4.1 PE modes, Thread mode and Handler mode

RCNMS There are two PE modes:
• Thread mode.
• Handler mode.

IFDVT A common usage model for the PE modes is:
Thread mode: Applications
Handler mode: OS kernel and associated functions. Software that manages system resources.

RRPKP The PE handles all exceptions in Handler mode.

RCMQP Thread mode is selected on reset. The IPSR is set to zero on reset.
