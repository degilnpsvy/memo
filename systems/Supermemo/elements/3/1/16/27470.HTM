VFPSmallRegisterBank()

Always TRUE for ARMv7-M. Indicates implementation of 16 doubleword registers

Pseudocode details of the FP extension registers on page A2-54


The pseudocode function VFPSmallRegisterBank() returns TRUE if an FP implementation provides access only to doubleword registers D0-D15. In an ARMv7-M implementation this function always returns TRUE.
