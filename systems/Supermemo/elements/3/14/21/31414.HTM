The NVIC_ISER<n> characteristics are:
Purpose: Enables or reads the enabled state of each group of 32 interrupts.
Usage constraints: Privileged access permitted only. Unprivileged accesses generate a BusFault.
    This register is word accessible only. Halfword and byte accesses are UNPREDICTABLE.
Configurations: This register is always implemented.
Attributes: 32-bit read/write-one-to-set register located at 0xE000E100 + 4n.
    Secure software can access the Non-secure view of this register via NVIC_ISER<n>_NS located at 0xE002E100 + 4n. The location 0xE002E100 + 4n is RES0 to software executing in Non-secure state and the debugger.
    This register is not banked between Security states.
