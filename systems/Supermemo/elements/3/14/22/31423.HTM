SHCSR, ICSR, DEMCR, NVIC_IABR<n>, NVIC_ISPR<n>, NVIC_ISER<n>, and STIR

SHCSR, System Handler Control and State Register, Provides access to the active and pending status of system exceptions.

ICSR, Interrupt Control and State Register, Controls and provides status information for NMI, PendSV, SysTick and interrupts.

DEMCR, Debug Exception and Monitor Control Register, Manages vector catch behavior and DebugMonitor handling when debugging.

NVIC_IABR<n>, Interrupt Active Bit Register, For each group of 32 interrupts, shows the active state of each interrupt.

NVIC_ISPR<n>, Interrupt Set Pending Register, Enables or reads the pending state of each group of 32 interrupts.

NVIC_ISER<n>, Interrupt Set Enable Register, Enables or reads the enabled state of each group of 32 interrupts.

STIR, Software Triggered Interrupt Register, Provides a mechanism for software to generate an interrupt.
