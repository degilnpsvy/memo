B5.13 Memory barriers

RWRCT The ARM architecture supports out-of-order completion of instructions.

RGKDW ARMv8 supports the following memory barriers:
• Instruction Synchronization Barrier (ISB).
• Data Memory Barrier (DMB).
• Data Synchronization Barrier (DSB).

RLQXF The DMB and DSB memory barriers affect reads and writes to the memory system that are generated by Load/Store instructions and data or unified cache maintenance instructions that are executed by the PE. Instruction fetches are not explicit accesses.
