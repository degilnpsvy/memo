B5.25 Cache identification

RWBGH A PE controls the implemented caches using:
• A single Cache Type Register, CTR.
• A single Cache Level ID Register, CLIDR.
• A single Cache Size Selection Register, CSSELR.
• For each implemented cache, across all levels of caching, a Cache Size Identification Register, CCSIDR.

RXJTL The number of levels of cache is IMPLEMENTATION DEFINED and can be determined from the Cache Level ID Register.

IPPSB Cache sets and Cache ways are numbered from 0. Usually the set number is an IMPLEMENTATION DEFINED function of an address.
