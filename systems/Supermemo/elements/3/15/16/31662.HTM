B5.17.3 Early Write Acknowledgement and no Early Write Acknowledgement Device memory attributes

E attribute

RPVSH This attribute imposes no restrictions or relaxations.

nE attribute

RFWFR Assigning the nE attribute recommends that only the endpoint of the write access returns a write acknowledgement of the access, and that no earlier point in the memory system returns a write acknowledgement.

IFQWQ The E attribute is treated as a hint. ARM strongly recommends that this hint is not ignored by a PE, but is made available for use by the system.
