D1.2.107 ICTR, Interrupt Controller Type Register

The ICTR characteristics are:
Purpose: Provides information about the interrupt controller.
Usage constraints: Privileged access permitted only. Unprivileged accesses generate a BusFault.
  This register is word accessible only. Halfword and byte accesses are UNPREDICTABLE.
Configurations: This register is always implemented.
Attributes: 32-bit read-only register located at 0xE000E004.
  Secure software can access the Non-secure view of this register via ICTR_NS located at 0xE002E004. The location 0xE002E004 is RES0 to software executing in Non-secure state and the debugger.
  This register is not banked between Security states.

The ICTR bit assignments are:

Bits [31:4]
Reserved, RES0.

INTLINESNUM, bits [3:0]
Interrupt line set number. Indicates the number of the highest implemented register in each of the NVIC control register sets, or in the case of NVIC_IPRn, 4×INTLINESNUM.
This field reads as an IMPLEMENTATION DEFINED value.
