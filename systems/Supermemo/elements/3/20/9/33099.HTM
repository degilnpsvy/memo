Shows or sets the value of the processor cycle counter, CYCCNT

The DWT_CYCCNT bit assignments are:

CYCCNT, bits [31:0]
Incrementing cycle counter value. Increments one on each processor clock cycle when DWT_CTRL.CYCCNTENA == 1 and DEMCR.TRCENA == 1. On overflow, CYCCNT wraps to
zero. This field resets to an UNKNOWN value on a Cold reset.
