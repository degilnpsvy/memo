2.10.1.6 bfd_check_overflow

Synopsis

     bfd_reloc_status_type bfd_check_overflow
        (enum complain_overflow how,
         unsigned int bitsize,
         unsigned int rightshift,
         unsigned int addrsize,
         bfd_vma relocation);

Description
Perform overflow checking on relocation which has bitsize significant bits and will be shifted right by rightshift bits, on a machine with addresses containing addrsizesignificant bits. The result is either of bfd_reloc_ok or bfd_reloc_overflow.
