Basics of the Tools

GNUPro Toolkit provides productivity, flexibility, performance and portability with its C and C++ compilers, its macro-assembler, its debugger (using command line or graphical user interfaces), its binary utilities, its libraries, and its other development tools. For a summary of the what GNUPro Toolkit provides, see “Compiler and Development Tools” on page 20, “Libraries” on page 20, and “Auxiliary Development Tools” on page 21.

To use the tools, in your system’s terminal shell window, enter their name as a command (gcc, for instance, invokes the C compiler); for working with the tools, see “Developing with GNUPro Toolkit” on page 37 and “Rebuilding from Source” on page 83.

To use the graphical user interface for the tools, enter the name as a command in your shell window (gdb invokes Insight and snavigator invokes Source-Navigator, for instance); see “Source-Navigator Demonstration” on page 63 and “Using Insight, the GUI Debugger” on page 74 for more information.
