Tools, Toolchains and Usage

A complete set of GNUPro tools for a particular native or host architecture and a target environment typically use a standard process for running, calling each tool with a toolchain name, a combination of a host name and a target name as a prefix to the tool name (if you are not already familiar with the names of specific tools, review “Compiler and Development Tools” on page 20, and “Auxiliary Development Tools” on page 21); see the following tables for the specific host and target names that you require.
