Configuring Cygmon

Use the following steps for configuring Cygmon. build_dir represents the complete path to your build directory. source_dir represents the complete path to the directory containing the source code.

1. Create a build directory and use the cd command to change to that directory.

cd build_dir

2. Configure the tools (use Table 5: “Target processors and naming conventions” on page 7 to replace canonicalname):

source_dir/configure —-target=canonicalname

3. Type the following command to build a Cygmon image in S-records that can be downloaded to a PROM burner or emulator.

make all-target-cygmon

The S-record image will be in the ~/cygmon directory.
