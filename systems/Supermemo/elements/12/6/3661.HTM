Building and Using DLLs with Cygwin

The following documentation provides an example of how to build a .dll file, using a single file, myprog.c, for the program, myprog.exe, and a single file, mydll.c, for the contents of the .dll file, mydll.dll, then compiling everything as objects. gcc -shared myprog.c -o mydll.dll -e _mydll_init@12 Now, when you build your program, you link against the import library, with declaration’s like the following example’s commands.

gcc myprog.o mydll.dll -o myprog.exe
