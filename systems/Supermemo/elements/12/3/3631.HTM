Compiler and Development Tools

The following tools are the main tools for developing projects with GNUPro Toolkit.

cpp

C preprocessor (see “cpp, the GNU Preprocessor” on page 23

 see also The C Preprocessor in GNUPro Compiler Tools)

diff

Comparison tools for text files (see Using diff& patch in GNUPro Development Tools)

 

diff3

sdiff

gcc

ISO-conforming C compiler (see “gcc and g++, the GNU Compilers” on page 22

 see also Using GNU CC in GNUPro Compiler Tools)

gcov

Coverage analyzer, for testing code for efficiency and performance, and for use as a profiling tool (see Using GNU CC in GNUPro Compiler Tools)

 

gdb -nw

Debugger for debugging with command line input (see “gdb, the Debugging Tool” on page 28

 see also Debugging with GDB in GNUPro Debugging Tools)

gdb

Debugger using a graphical user interface, a visual debugger, known as Insight and also conceptually known as gdbtk (see “Insight, a GUI Debugger” on page 29 and “Using Insight, the GUI Debugger” on page 74

 see also Insight, the GNUPro Debugger GUI Interface in GNUPro Debugging Tools)

g++

ISO-tracking C++ compiler (see “gcc and g++, the GNU Compilers” on page 22

 see also Using GNU CC in GNUPro Compiler Tools)

ld

Linker (see “ld, the GNU Linker” on page 24

 see also Using ld in GNUPro Development Tools)

make

Compilation control program (see “make, the GNU Recompiling Tool” on page 25

 see also Using make in GNUPro Utilities)

patch

Installation tool for source fixes (see Using diff& patch in GNUPro Development Tools)

 

snavigator

Source-Navigator, a source code comprehension tool (see “Source-Navigator, a Source Code Comprehension Tool” on page 31 and “Source-Navigator Demonstration” on page 63

 see also Source-Navigator’s online help documentation)
