Insight, a GUI Debugger

Besides the standard command line based debugger, gdb, GNUPro Toolkit includes Insight, a graphical user interface.Insight works on a range of host systems and target microprocessors, allowing development with complete access to the program state, for source and assembly level, with the ability to manage breakpoints, variables, registers, memory, threads and other functionality.

Adding a series of intuitive views into the debugging process, Insight provides you with a wide range of system information. See Figure 1 for an example of the windows that Insight provides for analyzing and debugging programs.
