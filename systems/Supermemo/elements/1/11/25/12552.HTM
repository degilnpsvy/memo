3.7.2.1 Linux host

As an example, you can download the ‘linux-test-xxx.tar.gz’ archive and copy the script ‘qemu-ifup’ in ‘/etc’ and configure properly sudo so that the command ifconfigcontained in ‘qemu-ifup’ can be executed as root. You must verify that your host kernel supports the TAP network interfaces: the device ‘/dev/net/tun’ must be present.

See Invocation to have examples of command lines using the TAP network interfaces.
