the CHECK_LIBRARY_EXISTS macro. It requires you to specify the library name, the name of a function provided by the library, a location to search, and a variable to store the result of the search. For example, to check for the dynamic loader library, use these commands:

INCLUDE(CheckLibraryExists) 
CHECK_LIBRARY_EXISTS(dl dlopen "" HAVE_LIBDL)

This checks for the dlopen() function in the libdl library and sets the HAVE_LIBDL variable to true if it’s found. CMake automatically checks standard system libraries, so an empty string suffices for the library location.
