GTAGS in Emacs

gtags-display-browser       C-c h

   Display current screen on hypertext browser.

gtags-find-file       C-c P

   Input pattern and move to the top of the file.

gtags-find-pattern       M-x ... RET

   Input pattern, search with grep(1) and move to the locations.

gtags-find-rtag       C-c r

   Input tag name and move to the referenced point.

gtags-find-symbol       C-c s

   Input symbol and move to the locations.

gtags-find-tag       C-c t, C-c d, M-.

   Input tag name and move to the definition.

gtags-find-tag-by-event       M-x ... RET

   Get the expression as a tagname around here and move there.

gtags-find-tag-from-here      C-]

   Get the expression as a tagname around here and move there.

gtags-find-tag-other-window   C-x 4 .

   Input tag name and move to the definition in other window.

gtags-find-with-grep       C-c g

   Input pattern, search with grep(1) and move to the locations.

gtags-find-with-idutils       C-c I

   Input pattern, search with idutils(1) and move to the locations.

gtags-make-complete-list      M-x ... RET

   Make tag name list for completion.

gtags-mode       M-x ... RET

   Toggle Gtags mode, a minor mode for browsing source code using GLOBAL.

gtags-parse-file       C-c f

   Input file name and show the list of tags in it.

gtags-pop-stack       M-*, C-t

   Move to previous point on the stack.

gtags-select-mode       M-x ... RET

   Major mode for choosing a tag from tags list.

gtags-select-tag       M-x ... RET

   Select a tag in [GTAGS SELECT MODE] and move there.

gtags-select-tag-by-event     M-x ... RET

   Select a tag in [GTAGS SELECT MODE] and move there.

gtags-select-tag-other-window M-x ... RET

   Select a tag in [GTAGS SELECT MODE] and move there in other window.

gtags-visit-rootdir       C-c v

   Tell tags commands the root directory of source tree.
