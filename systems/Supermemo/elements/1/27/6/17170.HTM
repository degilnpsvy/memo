Name

drm_framebuffer_unreference — unref a framebuffer
Synopsis
void drm_framebuffer_unreference ( struct drm_framebuffer * fb);
 
Arguments

fb

Description

This functions decrements the fb's refcount and frees it if it drops to zero.
