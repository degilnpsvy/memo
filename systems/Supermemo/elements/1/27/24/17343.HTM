drm_rect_intersect
Prev  Rectangle Utilities Reference  Next
Name

drm_rect_intersect — intersect two rectangles
Synopsis
bool drm_rect_intersect ( struct drm_rect * r1,
  const struct drm_rect * r2);
 
Arguments

r1

    first rectangle
r2

    second rectangle

Description

Calculate the intersection of rectangles r1 and r2. r1 will be overwritten with the intersection.
RETURNS

true if rectangle r1 is still visible after the operation, false otherwise.
