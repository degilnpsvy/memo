drm_dp_dpcd_readb
Prev  Display Port Helper Functions Reference  Next
Name

drm_dp_dpcd_readb — read a single byte from the DPCD
Synopsis
ssize_t drm_dp_dpcd_readb ( struct drm_dp_aux * aux,
  unsigned int offset,
  u8 * valuep);
 
Arguments

aux

    DisplayPort AUX channel
offset

    address of the register to read
valuep

    location where the value of the register will be stored

Description

Returns the number of bytes transferred (1) on success, or a negative error code on failure.
