For ar command, the options argument

l     r (replace): Insert an object file into the archive, replacing any previous object file of the same name. This is the standard method for creating and updating an archive. Thus, we might build an archive with the following commands:

1$ cc -g -c mod1.c mod2.c mod3.c

2$ ar r libdemo.a mod1.o mod2.o mod3.o

3$ rm mod1.o mod2.o mod3.o

As shown above, after building the library, we can delete the original object files if desired, since they are no longer required.
