Example program

Listing 42-1 demonstrates the use of the dlopen API. This program takes two commandline arguments: the name of a shared library to load and the name of a function to execute within that library. The following examples demonstrate the use of this program:

1$ ./dynload ./libdemo.so.1 x1

2Called mod1-x1

3$ LD_LIBRARY_PATH=. ./dynload libdemo.so.1 x1

4Called mod1-x1

In the first of the above commands, dlopen() notes that the library path includes a slash and thus interprets it as a relative pathname (in this case, to a library in the current working directory). In the second command, we specify a library search path in LD_LIBRARY_PATH. This search path is interpreted according to the usual rules of the dynamic linker (in this case, likewise to find the library in the current working directory).

 3int

 4main(int argc, char *argv[])

 5{

 6  void *libHandle; /* Handle for shared library */

 7  void (*funcp)(void); /* Pointer to function with no arguments */

 8  const char *err;

 9  if (argc != 3 || strcmp(argv[1], "--help") == 0)

10    usageErr("%s lib-path func-name\n", argv[0]);

11  /* Load the shared library and get a handle for later use */

12  libHandle = dlopen(argv[1], RTLD_LAZY);

13  if (libHandle == NULL)

14    fatal("dlopen: %s", dlerror());

15  /* Search library for symbol named in argv[2] */

16  (void) dlerror(); /* Clear dlerror() */

17  *(void **) (&funcp) = dlsym(libHandle, argv[2]);

18  err = dlerror();

19  if (err != NULL)

20    fatal("dlsym: %s", err);

21  /* If the address returned by dlsym() is non-NULL, try calling it

22     as a function that takes no arguments */

23  if (funcp == NULL)

24    printf("%s is NULL\n", argv[2]);

25  else

26    (*funcp)();

27  dlclose(libHandle); /* Close the library */

28  exit(EXIT_SUCCESS);

29}
