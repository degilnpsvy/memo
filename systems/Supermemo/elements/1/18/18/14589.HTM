42, ADVANCED FEATURES OF SHARED LIBRARIES

The previous chapter covered the fundamentals of shared libraries. This chapter describes a number of advanced features of shared libraries, including the following:

l     dynamically loading shared libraries;

l     controlling the visibility of symbols defined by a shared library;

l     using linker scripts to create versioned symbols;

l     using initialization and finalization functions to automatically execute code when a library is loaded and unloaded;

l     shared library preloading; and

l     using LD_DEBUG to monitor the operation of the dynamic linker.
