The STL is the most popular of these libraries, and it comes with all modern C++ compilers. The STL contains many useful data types and structures, including an improved character array known as string, and generic containers such as the vector<T>, map<T,T>, set<T>, list<T>, and deque<T>. These containers can be manipulated by STL algorithms such as for_each(), count(), min_element(), max_element(), search(), transform(), reverse(), and sort(). These are just a few of the algorithms available.
