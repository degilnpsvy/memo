Another view, show-branch, provides concise, one-line summaries for the current development branch:

$ git show-branch [...]

[master] Convert to HTML

[master^] Initial contents of public_html

The phrase [...] reveals up to an additional 10 more versions, but only two exist so far and so both are shown. (The default in this case would list only the most recent commit.) The name master is the default branch name.
