The address range that the VGA hardware decodes is based upon the Memory Map Select field. The following table shows the address ranges in absolute 32-bit form decoded for each value of this field:

  • 00 -- A0000h-BFFFFh -- 128K
  • 01 -- A0000h-AFFFFh -- 64K
  • 10 -- B0000h-B7FFFh -- 32K
  • 11 -- B8000h-BFFFFh -- 32K
