6.1. Motivation

Advantages of a simulator:

  • No need for special hardware that allows low-level programming. (We cannot write our own device drivers for miller and weise!)

  • Portability: We can change the platform used for programming in a course while continuing to use the same simulator.

  • We can more easily monitor the program while it runs, since a simulator has many capabilities that real hardware lacks.
