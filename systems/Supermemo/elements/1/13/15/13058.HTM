6.2. The Normal Assembly Process

  1. Edit the source code.

  2. Assemble (translate assembly language to machine language).

  3. Execute the machine language.

Native assembly language programming under Unix:

	miller: vi prog.s
	miller: as prog.s
	miller: a.out
