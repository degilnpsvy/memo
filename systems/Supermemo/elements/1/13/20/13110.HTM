12.11. Homework

  1. When should arrays be used? Why?

  2. What is the address of coefficients[12] in a Java array defined as double [] coefficients = new double[MAX_AGES];. A double occupies 64 bits. Assume the address of coefficients is 2000.

  3. What is the address of coefficients[3][4] in the C array int table[100][30];? Assume an int is 32 bits and the address of table is 4000. C uses row-major organization, and the first subscript is the row.

  4. Without looking at the notes, write a program that prints the factorials of all integers from 1 to 20. Use a factorial subprogram that employs a lookup table.

  5. Without looking at the notes, write a program that reads in a list of real numbers, and prints the list backwards. The first line of input is the size of the list. Each line after that contains one list element. Use dynamic memory allocation.

  6. Explain what happens when a program accesses a virtual memory address that is currently in swap.

  7. If RAM access takes 5ns, disk access takes 4ms, and the virtual memory hit ratio is 0.99, what is the average virtual memory access time in ns? How much of a slow-down is caused by swapping?

  8. A system has 4 gigabytes of RAM, and 6 gigabytes of swap. What is the virtual memory capacity of the system?

  9. A system has 1 gigabyte of main memory (RAM) and 8 megabytes of cache. What is the total RAM capacity of the system?

  10. If main memory access takes 10ns, cache access takes 2ns, and the cache hit-ratio is 0.78, what is the average RAM access time? What is the speedup provided by the cache in this case?

  11. What is locality of reference?

  12. How exactly does locality of reference affect program performance? How much difference can it really make?
