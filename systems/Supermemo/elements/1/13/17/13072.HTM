8.2. Arithmetic Expressions

Consider the following high level language statement:

	y = 3x^2 + 2x + 1

What does the compiler convert this to?

	# First term
	l.d     $f0, x
	li.d    $f2, 3.0
	mul.d   $f4, $f0, $f0   # $f4 = x^2
	mul.d   $f4, $f4, $f2   # $f4 = 3x^2
	
	# Second term
	li.d    $f2, 2.0
	mul.d   $f6, $f2, $f0   # $f6 = 2x
	
	# Sum
	li.d    $f2, 1.0
	add.d   $f4, $f4, $f6   # $f4 = 3x^2 + 2x
	add.d   $f4, $f4, $f2   # $f4 = 3x^2 + 2x + 1
	
	# Assignment
	s.d     $f4, y
