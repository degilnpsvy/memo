8.4. Switch Statements

8.4.1. If-else if-...-else

A switch statement can behave exactly like a series of if-else blocks:

Switch

	    switch(num)
	    {
		case    10:
		    statement1;
		    break;
		case    20:
		    statement2;
		    break;
		default:
		    statement3;
		    break;
	    }

If-else

	    if ( num == 10 )
		statement1;
	    else if ( num == 20 )
		statement2;
	    else
		statement3;

MAL

		lw      $t0, num
		bne     $t0, 10, else1
		# statement1
		j       end_switch
		
	    else1:
		bne     $t0, 20, else2
		# statement2
		j       end_switch
	    
	    else2:
		# statement3
	    
	    end_switch:
