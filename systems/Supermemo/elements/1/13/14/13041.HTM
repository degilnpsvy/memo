4.12. One's complement

4.12.1. Format

Positive values in one's complement are the same as unsigned binary or sign-magnitude.

4.12.2. Negation

To negate a one's complement value, we invert all bits. Like sign-magnitude, one's complement has two representations for zero (all zeros or all ones).

4.12.3. Conversions

What are the decimal values of the following 8-bit one's complement numbers?

  • 00001010 = +10
  • 10001010 = -(01110101) = -(1+4+16+32+64) = -117
  • 11111111 = ?

4.12.4. Addition and Subtraction

Addition and subtraction work like unsigned for positive numbers. If negative numbers are involved, arithmetic is slightly complicated. One's complement is not used for this reason.

4.12.5. Range

Range is the same as sign-magnitude.

4.12.6. Comparison

Comparison works like unsigned if both values have the same sign. If the signs differ, the positive number is greater.
