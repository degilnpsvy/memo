4.14. Biased Notation

4.14.1. Format

Biased notation stores a number N as an unsigned value N+B, where B is the bias. B is typically half the unsigned range, but doesn't have to be.

Bias-7 0 1 15 0000 0001 ... 1111 -7 -6 +8

8-bit biased-127 notation:

11111111 = 255 - 127 = +128

00000000 = 0 - 127 = -127

01111111 = 127 - 127 = 0

10000000 = ?

9 is stored as 127 + 9 = 136 = 10001000

-5 is stored as ?

Note that the bias does not have to be the mid-point. We can use any bias that can be represented in the number of bits available. Example: bias-3 4-bit: 0000 0001 ... 1111 -3 -2 +12

4.14.2. Negation

Subtract twice the value. E.g., to negate +6, subtract 12. To negate -6, subtract -12.
