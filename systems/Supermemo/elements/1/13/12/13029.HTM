3.4. Homework

  1. Define the following terms in the context of CPU hardware:

    • Machine language

    • Instruction set

    • CPU Architecture

    • Implementation

  2. Describe the basic structure of a machine instruction.

  3. Discuss three major limitations of machine language.

  4. Describe four advantages of programming in assembly language over programming in machine language.

  5. What do assembly language and machine language have in common?

  6. Describe two advantages of programming in a high-level language over programming in assembly language.

  7. Describe two major advantages of a compiled language over an interpreted language.

  8. What is a possible advantage of interpreted languages over compiled languages?

  9. Describe what happens when a compiled program is executed.

  10. Describe what happens when an interpreted program is executed.

  11. Rank the following languages in order of program execution speed.

    • C
    • C-shell
    • C++
    • Java (with JIT)
    • Perl

  12. What is the JIT compiler, and what does it do?

  13. Describe each of the stages of the software life cycle.

  14. How important are the specification and design stages? Why?

  15. Who is responsible for specifications? Why?

  16. What should be decided in the design stage?

  17. Describe the three major types of testing discussed in this chapter. What role does each play in the life cycle?
