2.6. Homework

You should be able to answer most of these questions based on the material in this chapter. For questions asking for a Unix command, be sure to verify your answers by trying them out on a real Unix system.

  1. How do you ensure that terminal-based programs like editors will work properly during your login session?

  2. What is the main difference between Unix and DOS/Windows text files?

  3. What is the root directory?

  4. What is a process? How can you list the processes you currently have running?

  5. What is the current working directory? What is a working directory always associated with?

  6. How can you see which processes are using the most resources?

  7. What is a full pathname?

  8. What is a relative pathname?

  9. What is shorthand for your home directory in a pathname?

  10. What is a shell?

  11. Explain the difference between internal and external commands.

  12. What is the command path? How can you change it?

  13. Show a single command to achieve each of the following:

    • List the files in /etc, showing the size and owner for each file.

    • View a file called prog1.c in a subdirectory of the current working directory, called Prog1.

    • Set the current working directory for the shell to your home directory.

    • Set the current working directory for the shell to a directory called Prog1, which is under your home directory.

    • List commands related to Java.

    • Remove all files in the current working directory whose names end in “.o”.

    • List the all the files in the current working directory, saving the output to a file called “list”.

    • List the all the files in the current working directory whose names end with “.java”, viewing the output no more than one screen at a time.

  14. What is a shell script?

  15. When should shell scripts be used, and how do they make you more productive?

  16. Describe each of the three standard streams, and what file or device are they normally bound to?

  17. How can the user change the standard streams for a given process?
