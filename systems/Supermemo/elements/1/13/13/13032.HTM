4.3. Numeric Limitations of Computers

Computers cannot represent infinite mathematical sets like the integers or reals, since they are infinite.

For integers, computers use something called modulo number systems, or modular arithmetic. A modulo number system is simply a subset of the integers ranging from 0 to some maximum value. The modulus is 1 more than the maximum value, so a modulo-50 system would allow values from 0 to 49. The modulus can be any positive integer, but usually corresponds to the maximum value that can be represented in a given number of digits. In decimal, we would likely use modulo-10 (0 to 9), modulo-100 (0 to 99), etc.

Example 4.1. A module-100 system

Modulo-100: A subset of the integers limited to two decimal digits. Values range from 00 to 99.

	    1       1
	    45      59
	+   27  +   62
	------  ------
	    72     121

In the case of 59 + 62, the modulo-100 result is 21. This is known as overflow.
