4.4. Fixed Point Number Systems

A fixed point decimal number system has a limited number of digits, and a decimal point in a fixed location.

Example 4.2. A fixed-point system

Suppose a system has 4 digits, two of which are fractional. Then all numbers have the form ##.##.

	    04.01       10.00       03.33
	+   12.58   /   03.00   *   03.00
	---------   ---------   ---------
	    16.59       03.33       09.99

The answer 03.33 is not precise. Note that reversing the operation does not produce the original value of 10.00. This is called round-off error.
