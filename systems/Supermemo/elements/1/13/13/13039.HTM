4.10. Unsigned Binary Integers

An unsigned binary integer is a fixed-point system with no fractional digits.

Unsigned binary integers are modulo number systems, usually with a modulus which is a power of 2.

Example 4.4. A 4-bit unsigned binary number system

A 4-bit unsigned binary number has values ranging from 00002 (010) to 11112 (1510). Hence, it is a modulo-100002, or modulo-1610.


Modern computers typically support binary integers of 8, 16, 32, or 64 bits.

The largest value in any unsigned binary integer system is the one containing all 1's, just as the largest decimal number is the one containing all 9's.

Largest modulo-100010 = 99910.

Largest modulo-10002 = 1112.

If we add 1 to a binary value of all 1's, note what happens:

	111
	1111
    +      1
    --------
       10000

If we add one to the largest value possible in 4 bits, we get 24. If we add 1 to the largest value possible in N bits, we will get 2N. Hence, the largest value possible in N bits, is 2N-1.

Table 4.2. Unsigned Binary Integer Ranges

Bits Range
8 0 to 28-1 (255)
16 0 to 216-1 (65,535)
32 0 to 232-1 (4,294,967,295)
64 0 to 264-1 (18,446,744,073,709,551,615 = 1.844674407 x 1019)

Note that even a 64-bit integer cannot hold Avogadro's constant, 6 x 1023. In order to represent very large numbers with only 32 or 64 bits, which must find a system other than fixed-point. This is discussed in Section 4.16, “Floating Point”.
