4.8. Converting to and from Decimal

4.8.1. Converting Other Bases to Decimal

Digits are limited to base-1. E.g. base 10 uses digits 0 to 9, base 8 uses 0 to 7, base 2 uses 0 to 1.

For bases larger than 10, we use letters: A = 10, B = 11, ...

Base 16 (hexadecimal, or hex for short) uses digits 0 to 9 and A to F.

1001.112 = 1 * 23 + 1 * 20 + 1 * 2-1 + 1 * 2-2 = 8 + 1 + 1/2 + 1/4

34.58 = 3 * 81 + 4 * 80 + 5 * 8-1 = 24 + 4 + 5/8

3F.816 = 3 * 161 + 15 * 160 + 8 * 16-1 = 48 + 15 + 8/16

The Double-and-Add Method

Binary integers can be converted easily by scanning left to right, doubling the value so far for each new digit (whether 0 or 1) and adding the new digit to the sum.

	    1101
	    
	    1)  0 + 1 = 1
	    2)  2 + 1 = 3
	    3)  6 + 0 = 6
	    4)  12 + 1 = 13

The Remainder Method

We can quickly convert decimal to other bases by repeatedly dividing by the base (using integer division) and taking the remainder as the next digit (from right to left).

	    Decimal to binary
	    
	    37      = 100101
	    
	    18 r1
	     9 r0
	     4 r1
	     2 r0
	     1 r0
	     0 r1
	    
	    Decimal to octal
	    
	    135     = 207
	    
	     16 r7
	      2 r0
	      0 r2
	    
	    Decimal to hexadecimal
	    
	    135     = 87
	    
	     8 r7
	     0 r8
