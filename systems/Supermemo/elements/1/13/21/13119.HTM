14.5. Homework

  1. What are I/O devices?

  2. What does an I/O device look like from the perspective of the CPU?

  3. Describe the types of registers found in I/O controllers.

  4. Draw a diagram showing the connections between the CPU and an I/O device.

  5. Explain the difference between memory-mapped I/O and separate I/O.

  6. Write a MAL program that inputs a character from the keyboard using spin-waiting, and outputs the same character to the display using spin-waiting.

  7. Explain the difference between spin-waiting and periodic polling.

  8. What are the advantages and disadvantages of spin-waiting?

  9. What are the drawbacks of periodic polling? What is the solution?

  10. Explain the difference between blocking and non-blocking I/O. What kinds of program use each?
