12.1. Motivation

Understand how HLL arrays and structures are organized in memory. Effects of program structure on memory hierarchy.

Computer memory is essentially a 1-dimensional array of cells, with each cell containing 1 byte on most systems. All data structures such as arrays, structures, classes, etc. must be mapped into this linear address space.
