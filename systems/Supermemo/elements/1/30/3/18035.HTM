B.3.1.6 Using gawk In The MSYS Environment

In the MSYS environment under MS-Windows, gawk automatically uses binary mode for reading and writing files. Thus there is no need to use the BINMODE variable.

This can cause problems with other Unix-like components that have been ported to MS-Windows that expect gawk to do automatic translation of "\r\n", since it won’t. Caveat Emptor!
