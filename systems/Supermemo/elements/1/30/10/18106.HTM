NAME

vga_getdefaultmode - returns the default graphics mode number  

SYNOPSIS

#include <vga.h>

int vga_getdefaultmode(void);

 

DESCRIPTION

Returns the default graphics mode number from the SVGALIB_DEFAULT_MODE environment variable; -1 if undefined. The environment variable can either be a mode number or a mode name. See vga_setmode(3) for a list of predefined return values and their meaning. The mode names which can be used in the environment are those used by vga_getmodename(3) and vga_getmodenumber(3).

Whenever your application has no special need for a given mode you should always try to honour this setting for consistency among svgalib applications.
