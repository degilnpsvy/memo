NAME

vga_claimvideomemory - declare the amount of video memory used  

SYNOPSIS

#include <vga.h>

int vga_claimvideomemory(int m);

 

DESCRIPTION

Declare the amount of video memory in bytes that is to be saved when temporarily switching to textmode. Returns nonzero if the amount is not available. Defaults to one screen when a mode is set. Use this if you are using more than one screen of video memory.
