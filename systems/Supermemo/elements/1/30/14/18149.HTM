NAME

mouse_setposition - set the current mouse position  

SYNOPSIS

#include <vgamouse.h>

void mouse_setposition(int x, int y);

 

DESCRIPTION

Sets the current mouse position to (x, y).

For 3d pointer devices, there is actually another interface. See mouse_getposition_6d(3).
