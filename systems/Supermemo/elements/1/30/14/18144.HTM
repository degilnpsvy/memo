NAME

mouse_waitforupdate - wait for an mouse update  

SYNOPSIS

#include <vgamouse.h>

void mouse_waitforupdate(void);

 

DESCRIPTION

This reads in mouse events from the mouse buffer and updates the internal svgalib variables which store the current mouse position and button states. If there are no events the function does not return until one occurs.
