NAME

vga_getpalette - get a color in the color lookup table  

SYNOPSIS

#include <vga.h>

void vga_getpalette(int index, int *r, int *g, int *b);

 

DESCRIPTION

Get the color with index index and store its RGB values at *r, *g, and *b, each in the range 0 - 63. This function is only sensible in 16 or 256 color modes. It returns which color all vga_setcolor(index) drawn pixels actually refer to.

If the VGA_CLUT8 feature is set for the current mode with vga_ext_set(3) the *r, *g, and *b values are in the range 0 - 255.
