As a convenience, you can define a variable and export it at the same time by doing: 
     export variable = value

has the same result as: 

     variable = value
     export variable

and 

     export variable := value

has the same result as: 

     variable := value
     export variable

Likewise, 

     export variable += value

is just like: 

     variable += value
     export variable

See Appending More Text to Variables.
