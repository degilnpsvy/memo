16.6 Example: Some File Functions

No matter where you go, there you are.

— Buckaroo Bonzai

Two useful functions that are not in awk are chdir() (so that an awk program can change its directory) and stat() (so that an awk program can gather information about a file). This section implements these functions for gawk in an extension.
