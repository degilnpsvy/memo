The tricky part comes into play when you need to work with indexed arrays values. If the index value already exists, it’s easy to replace with a new value:

$ echo $path

/usr/local/bin /bin /usr/bin /usr/X11R6/bin

$ set path[4]=/home/test

$ echo $path

/usr/local/bin /bin /usr/bin /home/test

$

However, you can’t use this technique to add a new element:

$ set path[5]=/home/test

set: Subscript out of range.

$

Instead, you must use parentheses to denote the new array: use $path to insert the existing array elements into the new array, then add the new array elements at the end:

$ set path[...]

$ echo $path

/usr/local/bin /bin /usr/bin /home/test /home/test2

$ echo $path[5]

/home/test2

$

Now the new array value contains the additional element.
