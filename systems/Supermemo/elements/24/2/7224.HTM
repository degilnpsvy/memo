Table 7-3 lists the commands used to center text.

Table 7-3. Centering commands

Keystrokes     Command name    Action

M-o M-s                 center-line            Center the line the cursor is on.

M-o M-S                center-paragraph       Center the paragraph the cursor is on.

(none)                 center-region                Center the currently defined region.

(none)Edit Text Properties Justification Center     set-justification-center        Center selected text.
