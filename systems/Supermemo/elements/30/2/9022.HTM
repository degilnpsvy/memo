2.16.1.4 bfd_open_file

Synopsis

     FILE* bfd_open_file (bfd *abfd);

Description
Call the OS to open a file for abfd. Return the FILE * (possibly NULL) that results from this operation. Set up the BFD so that future accesses know the file is open. If theFILE * returned is NULL, then it won't have been put in the cache, so it won't have to be removed from it.
