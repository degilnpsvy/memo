6.12. getpid

This function returns the context's process id, which we can emulate using uC/OS's OSTaskQuery() function, as we did for __malloc_lock. The code is in Figure 17.

Figure 17. The getpid stub.

int getpid ( void ) { OS_TCB tcb; INT8U err; int id;

/* use our priority as a task id */ OSTaskQuery( OS_PRIO_SELF, &tcb ); id = tcb.OSTCBPrio;

return id; }
