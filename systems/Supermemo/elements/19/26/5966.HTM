2. Newlib Licenses

Newlib is a collection of source code, distributed under the terms of several different public-domain and BSD-style licenses. Proprietary embedded applications can therefore use newlib, because its licenses do not require distribution of the final application's source code. Newlib was assembled and is distributed by Cygnus Solutions, Inc., now a Red Hat, Inc. company.

All of newlib's licenses are gathered into the file COPYING.NEWLIB, which is included with the newlib source code. You should read this file before you decide to use newlib, the same way you do with the license for any other software you use.
