l         Repo forall

Perform operation for specific projects list

$ repo forall [proj_list] –c command

Environment variables

REPO_PROJECT, REPO_PATH, REPO_REMOTE, REPO_LREV, REPO_RREV

Example – check all project revisions (branch)

$ repo forall –pc ‘echo $REPO_RREV’

Example – reset & clean all repositories

$ repo forall –c git reset –hard

$ repo forall –c git clean –f –d

Repo-Based Development

Workflow for Large Project

l         A feature may across multiple repositories

Regular branch name are easy to manage

EX: feature/<name>, hotfix/<bug>, release/<ver>

Use manifest files to management various objectives of development

$ repo init -m manifest.xml

Ex: feature-<name>.xml, hotfix-<bug>.xml

l         Managers take care of main branches

Feature-integration & product-release

l         Developers take care of supporting branches

 

l         Work on a Feature Development

l         Switch to feature branches

$ repo init -m feature/<name>.xml

$ repo sync

l         Work on topic branch

$ repo start <branch-name> <proj_list>

Make and upload some great git commits

$ repo upload [proj_list]

After code review

$ repo sync && repo prune

 

l         Prepare a Feature Branch

l         Create & push feature branch (or via Gerrit UI)

$ repo forall -c git checkout -b feature/<name> origin/dev/<name>

$ repo forall -c git push origin feature/<name>

 

l         Edit & push feature-<name>.xml

Use repo to add manifest files, or

Use git to update manifest repository

 

l         Finish a Feature Branch

l         Switch to main branch

$ repo init -m dev/<name>.xml; repo sync

l         Start topic branch to merging feature-branch

$ repo start dev/<name>

$ repo forall -c git merge --no-ff origin/feature/<name>

$ repo upload

 

l         Delete merged feature-branch (or via Gerrit UI)

$ repo forall -c git push origin : feature/<name>

 

l         Add Tag to Released Commit

l         Add annotated tag to production-ready commit

$ repo forall -c git tag <version>

l         Push tag to centralized repository

$ repo forall -c git push origin tag <version>

l         To delete a remote tag

$ repo forall -c git push origin :refs/tags/<version>
