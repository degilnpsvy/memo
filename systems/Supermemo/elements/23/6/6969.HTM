The pathname [...] is normally a symbolic link pointing to the dynamic linker executable file. This file has the name ld-version.so, where version is the glibc version installed on the system—for example, ld-2.11.so. The pathname of the dynamic linker differs on some architectures. For example, on IA-64, the dynamic linker symbolic link is named /lib/ld-linux-ia64.so.2.
