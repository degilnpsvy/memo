l         Repository Management

l         Add a repository

$ git remote add <name> <url>

l         Remove a repository

$ git remote rm <name>

l         Show available repository

$ git remote show <name>

 

Fetch Repository Branches

l         Fetch branches

$ git fetch <remote>

Example

$ git fetch official

$ git branch –r

 

l         Push/Pull Commits

l         Push local branch to remote

$ git push (permission required)

l         Pull remote branch

$ git pull

May have conflicts to resolve

Repo uses `repo sync/upload’ instead

 

l         Managing Remote Branches

l         Require write permission to remote repository

$ man git-config

l         Push local branch to remote repository

$ git push <origin> <branch>

l         Delete remote branch

$ git push <origin> :<branch>
