A.2 Add-on packages

A large number of add-on packages have been written by various authors.

These packages are not part of Emacs, but they are distributed as contributed packages with the separate release available at http://orgmode.org. See the contrib/READMEfile in the source code directory for a list of contributed files. You may also find some more information on the Worg page: http://orgmode.org/worg/org-contrib/.
