14.8.2.17 :sep

The :sep header argument can be used to control the delimiter used when writing tabular results out to files external to Org mode. This is used either when opening tabular results of a code block by calling the org-open-at-point function bound to C-c C-o on the code block, or when writing code block results to an external file (seefile) header argument.

By default, when :sep is not specified output tables are tab delimited.
