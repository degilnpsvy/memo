1748/**
1749 * wake_up_process - Wake up a specific process

1753 * processes.
1754 *
1755 * Return: 1 if the process was woken up, 0 if it was already running.
1756 *
1757 * It may be assumed that this function implies a write memory barrier before

1759 */
