SDL_TryLockMutex

Use this function to try to lock a mutex without blocking.

Syntax

切换行号显示
int SDL_TryLockMutex(SDL_mutex* mutex)

Function Parameters

mutex

the mutex to try to lock

Return Value

Returns return 0, SDL_MUTEX_TIMEDOUT, or -1 on error; call SDL_GetError() for more information.

Code Examples

切换行号显示
   1 int status;
   2 SDL_mutex *mutex;
   3 
   4 mutex = SDL_CreateMutex();
   5 if (!mutex) {
   6   fprintf(stderr, "Couldn't create mutex\n");
   7   return;
   8 }
   9 
  10 status = SDL_TryLockMutex(mutex);
  11 
  12 if (status == 0) {
  13   printf("Locked mutex\n");
  14   SDL_UnlockMutex(mutex);
  15 } else if (status == SDL_MUTEX_TIMEDOUT) {
  16   /* Mutex not available for locking right now */
  17 } else {
  18   fprintf(stderr, "Couldn't lock mutex\n");
  19 }
  20 
  21 SDL_DestroyMutex(mutex);
