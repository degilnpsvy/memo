SDL_CreateCond

Use this function to create a condition variable.

Syntax

切换行号显示
SDL_cond* SDL_CreateCond(void)

Return Value

Returns a new condition variable or NULL on failure; call SDL_GetError() for more information.

Code Examples

Typical use of condition variables:

切换行号显示
   1 SDL_bool condition = SDL_FALSE;
   2 SDL_mutex *lock;
   3 SDL_cond *cond;
   4 
   5 lock = SDL_CreateMutex();
   6 cond = SDL_CreateCond();
   7 .
   8 .
   9 Thread A:
  10     SDL_LockMutex(lock);
  11     while (!condition) {
  12         SDL_CondWait(cond, lock);
  13     }
  14     SDL_UnlockMutex(lock);
  15 
  16 Thread B:
  17     SDL_LockMutex(lock);
  18     ...
  19     condition = SDL_TRUE;
  20     ...
  21     SDL_CondSignal(cond);
  22     SDL_UnlockMutex(lock);
  23 .
  24 .
  25 SDL_DestroyCond(cond);
  26 SDL_DestroyMutex(lock);
