The userspace command-line utility dmidecode(8) inspects this data (SMBIOS data). Information provided by this utility typically includes the system manufacturer, model name, serial number, BIOS version and asset tag, as well as a lot of other details of varying level of interest and reliability what depends on the system manufacturer. The information often includes usage status for the CPU sockets, expansion slots (including AGP, PCI and ISA) and memory module slots, and the list of I/O ports (including serial, parallel and USB).[1][2]
