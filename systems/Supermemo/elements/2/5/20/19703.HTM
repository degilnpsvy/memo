除了exec，输入输出重定向和read命令配合也很好用，read命令每次读取文件的一行。但是要注意的是，输入输出重定向放到for、while等循环的循环体和循环外，效果是不一样的。如下图：

#!/bin/bash

 

for i in 1 2 3 4 5; do echo "This is line $i"; done > temp_file # temp file for tests

while read line

do

    echo $line

done < temp_file

 

while read line < temp_file

do

    echo $line

done

 

$ ./test.sh

This is line 1

This is line 2

This is line 3

This is line 4

This is line 5

This is line 1

This is line 1

This is line 1

This is line 1

This is line 1
