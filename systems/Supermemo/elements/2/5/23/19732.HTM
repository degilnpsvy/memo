Opening a Window

We’ll need a window to display our render in, we can create one with SDL_CreateWindow which takes a title for the window, the x and y position to create it at, the window width and height and some flags to set properties of the window and returns anSDL_Window*. This pointer will be NULL if anything went wrong when creating the window. If an error does occur we need to clean up SDL before exiting the program.
