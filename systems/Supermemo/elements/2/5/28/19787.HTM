SDL_SetWindowTitle

Use this function to set the title of a window.

Syntax

切换行号显示
void SDL_SetWindowTitle(SDL_Window* window,
                        const char* title)

Function Parameters

window

the window to change

title

the desired window title in UTF-8 format
