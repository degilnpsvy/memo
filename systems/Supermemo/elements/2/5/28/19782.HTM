SDL_SetWindowIcon

Use this function to set the icon for a window.

Syntax

切换行号显示
void SDL_SetWindowIcon(SDL_Window*  window,
                       SDL_Surface* icon)

Function Parameters

window

the window to change

icon

an SDL_Surface structure containing the icon for the window
