ARMv8 has two Execution States

l  AArch32

n  A32 (ARM) and T32 (Thumb) ISAs

n  Traditional ARM exception models

n  Virtual addresses stored in 32-bit registers

n  LPAE to support 40-bit physical addressing

l  AArch64

n  New A64 ISA (but still 32-bit long)

n  New 64-bit GPRs (X0 to X30)

n  New exception model

n  Virtual addresses stored in 64-bit registers

n  Up to 48-bit virtual/physical addressing
