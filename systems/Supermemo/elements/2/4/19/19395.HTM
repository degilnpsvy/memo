SDL_BlendMode

An enumeration of blend modes used in SDL_RenderCopy() and drawing operations.

Values

SDL_BLENDMODE_NONE

no blending

dstRGBA = srcRGBA

SDL_BLENDMODE_BLEND

alpha blending

dstRGB = (srcRGB * srcA) + (dstRGB * (1-srcA))

dstA = srcA + (dstA * (1-srcA))

SDL_BLENDMODE_ADD

additive blending

dstRGB = (srcRGB * srcA) + dstRGB

dstA = dstA

SDL_BLENDMODE_MOD

color modulate

dstRGB = srcRGB * dstRGB

dstA = dstA
