Remarks

By default the APPLICATION category is enabled at the SDL_LOG_PRIORITY_INFO level and all other categories are enabled at the SDL_LOG_PRIORITY_CRITICAL level. See SDL_LogPriorityfor details.

SDL_LOG_CATEGORY_CUSTOM is reserved for application use; for example:

  • enum {
      MYAPP_CATEGORY_AWESOME1 = SDL_LOG_CATEGORY_CUSTOM,
      MYAPP_CATEGORY_AWESOME2,
      MYAPP_CATEGORY_AWESOME3,
      ...
};
