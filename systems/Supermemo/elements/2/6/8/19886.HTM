Remarks

An SDL_Point defines single two dimensional point. It can be used not only for points, but also for size. SDL_Point is used bySDL_EnclosePoints() to check if array of points is inside rectangle (SDL_Rect). You can also make your own functions using SDL_Point to simplify your code, it's very helpful.
