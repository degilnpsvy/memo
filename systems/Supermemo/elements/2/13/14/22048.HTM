Name

rcu_is_watching — see if RCU thinks that the current CPU is idle

Synopsis

bool notrace rcu_is_watching ( void);
 

Arguments

void

no arguments

Description

If the current CPU is in its idle loop and is neither in an interrupt or NMI handler, return true.
