work_on_cpu
Prev  Workqueues and Kevents  Next

Name

work_on_cpu — run a function in user context on a particular cpu

Synopsis

long work_on_cpu ( int cpu,
  long (*fn) (void *),
  void * arg);
 

Arguments

cpu

the cpu to run on

fn

the function to run

arg

the function arg

Description

It is up to the caller to ensure that the cpu doesn't go offline. The caller must not hold any locks which would prevent fn from completing.

Return

The value fn returns.
