hrtimer_start_range_ns
Prev  High-resolution timers  Next

Name

hrtimer_start_range_ns — (re)start an hrtimer on the current CPU

Synopsis

int hrtimer_start_range_ns ( struct hrtimer * timer,
  ktime_t tim,
  unsigned long delta_ns,
  const enum hrtimer_mode mode);
 

Arguments

timer

the timer to be added

tim

expiry time

delta_ns

"slack" range for the timer

mode

expiry mode: absolute (HRTIMER_MODE_ABS) or relative (HRTIMER_MODE_REL)

Returns

0 on success 1 when the timer was active
