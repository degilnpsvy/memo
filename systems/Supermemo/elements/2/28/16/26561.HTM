A7.7.228 VFNMA, VFNMS

Floating-point Fused Negate Multiply Accumulate negates one single-precision register value and multiplies it by another single-precision register value, adds the negation of the floating-point value in the destination single-precision register to the product, and writes the result back to the destination single-precision register. The result of the multiply is not rounded before the addition.

Floating-point Fused Negate Multiply Subtract multiplies together two single-precision register values, adds the negation of the floating-point value in the destination single-precision register to the product, and writes the result back to the destination single-precision register. The result of the multiply is not rounded before the addition.
