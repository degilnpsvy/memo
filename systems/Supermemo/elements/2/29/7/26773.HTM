Case 范围

GNU C 允许在一个 case 标号中指定一个连续范围的值，例如：

++++ arch/i386/kernel/irq.c
1062:                         case ‘0′ … ‘9′: c -= ‘0′; break;
1063:                         case ‘a’ … ‘f’: c -= ‘a’-10; break;
1064:                         case ‘A’ … ‘F’: c -= ‘A’-10; break;

case ‘0′ … ‘9′:

相当于

case ‘0′: case ‘1′: case ‘2′: case ‘3′: case ‘4′:
case ‘5′: case ‘6′: case ‘7′: case ‘8′: case ‘9′:
