The alignment requirement of the output section is the strictest alignment of any input section contained within the output section. For example .text . : { *(.text) } and .text : { *(.text) } are subtly different. The first will set the address of the .text output section to the current value of the location counter. The second will set it to the current value of the location counter aligned to the strictest alignment of a .text input section. The address may be an arbitrary expression.
